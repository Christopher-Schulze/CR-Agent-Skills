# Golden Quality Examples Reference

Use this for taste calibration when "premium", "sophisticated", or "high-end" is too vague. These are compact judgment examples, not copy templates.

## Contents

- Dashboard/Operations Tool
- Editor/Canvas Tool

- Universal Agency-Grade Test

## Landing/Product Page

- Weak: centered H1, abstract gradient, three feature cards, fake logos, generic CTA.
- Acceptable: clear offer, product mockup, specific benefit copy, clean spacing, coherent palette.
- Agency-grade: first viewport is built around the product's real workflow/result artifact; one memorable object or environment proves value; copy is short and concrete; downstream section rhythm varies density and media; every CTA/control is crafted.

## Dashboard/Operations Tool

- Weak: hero-like header, KPI cards, decorative chart tiles, random badges, no selected state.
- Acceptable: sidebar, table, filters, status colors, detail panel, useful row data.
- Agency-grade: dominant work shape is obvious within seconds; rows, statuses, filters, and selected detail encode the actual triage/approval/monitoring behavior; density is high but calm; component typography and state treatment make scanning faster than reading prose.

## Editor/Canvas Tool

- Weak: generic SaaS shell wrapped around a fake canvas screenshot.
- Acceptable: toolbar, canvas, inspector, layers, selected object, export action.
- Agency-grade: the work product owns visual attention; chrome is compact and precise; inspector fields, layer rows, handles, mode buttons, status bar, and export flow all feel like one crafted tool; default zoom and selected object are intentional.

## Mobile Transaction

- Weak: stacked marketing copy, tiny controls, fake trust badges, CTA below reach.
- Acceptable: one-step screen, clear inputs, primary action, validation, summary.
- Agency-grade: one decision per screen; tactile selector/progress/receipt object reduces uncertainty; validation and confirmation feel calm and recoverable; bottom action is reachable; every input and state looks premium without reducing legibility.

## Game Surface

- Weak: decorative background, unclear player/hazards, generic canvas shapes, HUD clutter.
- Acceptable: readable player, obstacles, score, restart, basic movement.
- Agency-grade: loop is clear before decoration; player, hazards, reward, goal, route, and HUD are instantly legible; generated art assets match collision geometry; motion and controls make the playfield feel alive without hiding gameplay.

## Universal Agency-Grade Test

- Hide the copy. The surface should still reveal product category, workflow, or world through layout, objects, data shape, components, and assets.
- Remove the main image. The typography, spacing, and components should still feel authored, not default.
- Inspect the smallest repeated control. It should feel as designed as the hero.
- Shrink to mobile. Priority, text wraps, CTA reach, and component craft should survive.
- Ask what a top design agency would fix in 20 minutes. If the answer is obvious and feasible, fix it before final.
